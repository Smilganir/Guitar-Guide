# Guitar note samples (optional)

Drop **OGG** files here to play real guitar notes instead of synthesized sound.

## File names

One file per note, named exactly: `{Note}{Octave}.ogg`

- Use `s` for sharp: `Cs3.ogg` = C#3, `Fs4.ogg` = F#4
- Use `b` for flat if needed: `Bb2.ogg` = B♭2

## Notes used by the app (standard tuning)

Strings (low to high): E2, A2, D3, G3, B3, E4 and frets 0–4.

Examples: `E2.ogg`, `A2.ogg`, `D3.ogg`, `G3.ogg`, `B3.ogg`, `E4.ogg`, `Fs2.ogg`, `Cs3.ogg`, etc.

## Where to get samples (CC0 / free)

- **FreePats** Spanish classical guitar (CC0):  
  https://freepats.zenvoid.org/Guitar/acoustic-guitar.html  
  Download the SFZ+WAV pack, extract, then convert WAV → OGG and rename to the scheme above.

- **Freesound** (check license per pack):  
  Search for "classical guitar single note" or "acoustic guitar chromatic".

If no file exists for a note, the app falls back to synthesized sound.
